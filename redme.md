1. Повторение базовых вещей (2 минуты)
– Что такое список (list) → последовательность данных.
– Что такое словарь (dict) → пары «ключ: значение».
– Почему перевод удобнее хранить в словаре, а не в списке.
Пример:
dictionary = {
    "привет": "hello",
    "кот": "cat",
    "собака": "dog"
}

— Ключ: русское слово.
— Значение: английский перевод.

2. Простейший переводчик через dict (4–5 минут)
Пишем код:
dictionary = {
    "привет": "hello",
    "кот": "cat",
    "яблоко": "apple",
    "молоко": "milk"
}

word = input("Введите слово на русском: ").lower()

if word in dictionary:
    print("Перевод:", dictionary[word])
else:
    print("Такого слова нет в словаре ")

Разбор:
.lower() — для слов без учёта регистра.
Проверка if word in dictionary
Обработка случая, когда слово не найдено.

3. Добавление новых слов пользователем (2–3 минуты)
Чтобы словарь “жил”, можно добавить функцию добавления:
dictionary = {}

while True:
    word = input("Введите слово на русском (или стоп): ").lower()
    if word == "стоп":
        break

    eng = input("Введите перевод: ").lower()
    dictionary[word] = eng

print(dictionary)

Разбор:
while True — бесконечный цикл.
break — выход.
Добавление в словарь.

4. Переводчик через библиотеку googletrans(опционально — если хочешь) (3–4 минуты)
! Не ИИ, а обычный API-библиотека.
Установка:
pip install googletrans==4.0.0-rc1

Код:
from googletrans import Translator

translator = Translator()

text = input("Введите слово или фразу: ")
result = translator.translate(text, src="ru", dest="en")

print("Перевод:", result.text)

Поговорить:
как работает API,
чем он отличается от ИИ,
какие ограничения.

5. Расширенная практика: мини-словарик + меню (5–6 минут)
Делаем мини-консольное приложение с меню:
1 — перевести слово
2 — добавить слово
3 — показать словарь
0 — выход

Пример:
dictionary = {
    "дом": "house",
    "рыба": "fish"
}

while True:
    print("\nМеню:")
    print("1 — Перевести слово")
    print("2 — Добавить слово")
    print("3 — Показать словарь")
    print("0 — Выход")

    choice = input("Ваш выбор: ")

    if choice == "1":
        word = input("Введите русское слово: ")
        print(dictionary.get(word, "Нет такого слова"))

    elif choice == "2":
        ru = input("Русское слово: ")
        en = input("Английский перевод: ")
        dictionary[ru] = en
        print("Слово добавлено!")

    elif choice == "3":
        print(dictionary)

    elif choice == "0":
        print("Выход.")
        break

Это уже мини-приложение уровня junior-practice.
